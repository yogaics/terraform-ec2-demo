# terraform-ec2-demo

How to use:
1. Tambahkan credential AWS di terraform.tfvars
2. Jalankan :

terraform init

terraform plan

terraform apply

3. Kalo sudah selesai testing, bisa dihapus dengan

terraform destroy

Untuk step-step ada di https://testajah.com/2022/08/19/melakukan-deployment-ec2-menggunakan-terraform/
